About The Project

Introduction

Estimating an object's precise 3D rotation and translation (6D Pose) from an image is a fundamental challenge in computer vision, with applications in robotics, augmented reality, and autonomous systems.

Target Pipeline

The process involves analyzing the target object to create a robust model for later identification.

Query Pipeline

The Query Pipeline takes a new image as input and processes it to find features that can be matched against the target model.

Pose Estimation

The Pose Estimation stage compares the features from the query and target to calculate the precise 6D pose.
